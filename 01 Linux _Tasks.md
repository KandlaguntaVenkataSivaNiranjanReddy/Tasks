# Linux Lab - Tasks

---

## Task 1: System Exploration and Navigation

### Objective
Understand how to move around the Linux system and check basic system details as a normal user.

### Requirements
- Find and display your current working directory  
- List all files, including hidden files, in the current location  
- Go to the `/tmp` directory and check what files are present  
- Come back to your home directory  
- Check which user is currently logged in  
- Display basic system information such as OS and kernel details  

---

## Task 2: Project Workspace Initialization

### Objective
Create a proper folder structure to simulate a real DevOps project setup.

### Requirements
- Create a directory named `linux_lab`  
- Inside it, create another directory called `day1`  
- Inside `day1`, create three files: `a.txt`, `b.txt`, and `c.txt`  
- Create a nested directory structure like `project/app/logs`  
- Display the complete directory structure in a tree format  

---

## Task 3: File Creation and Content Management

### Objective
Learn how to create files, add content, and verify the data inside them.

### Requirements
- Create a new file and add some sample text into it  
- Append additional text to the same file without removing old content  
- View the file content using different commands  
- Edit the file using a text editor and modify the content  
- Verify that the changes are saved correctly  

---

## Task 4: File Operations (Copy, Move, Rename)

### Objective
Perform common file operations used in real-world scenarios.

### Requirements
- Copy a file and create a backup of it  
- Move a file from one directory to another  
- Rename an existing file  
- Verify that all operations are completed successfully  

---

## Task 5: System Cleanup and Maintenance

### Objective
Learn how to remove unnecessary files and keep the workspace clean.

### Requirements
- Delete a specific file from a directory  
- Remove a directory along with all its contents  
- Confirm that the files and directories are removed properly  

---

## Task 6: File Permissions Management

### Objective
Understand how to control who can read, write, and execute files.

### Requirements
- Create a new file  
- Set very restrictive permissions on the file  
- Temporarily give full permissions to all users  
- Change the permissions back to a secure state  
- Verify the permission changes  

---

## Task 7: File Ownership Management

### Objective
Learn how to manage file ownership in a multi-user environment.

### Requirements
- Create a new user in the system  
- Assign the ownership of a file to that user  
- Verify that the ownership has been changed correctly  

---

## Task 8: File Search and Content Inspection

### Objective
Find files and search for specific content inside them.

### Requirements
- Search for all `.txt` files in a directory  
- Find files that contain a specific keyword  
- Display only the lines that match the keyword  

---

## Task 9: File Viewing and Monitoring

### Objective
Learn how to read and monitor file content efficiently.

### Requirements
- Display the first few lines of a file  
- Display the last few lines of a file  
- Monitor a file in real time as new data is added  
- Add new content and observe how it updates live  

---

## Task 10: Data Sorting and Deduplication

### Objective
Organize and clean data using sorting and duplicate removal techniques.

### Requirements
- Create a file with unsorted data  
- Sort the data in alphabetical order  
- Sort the data in reverse order  
- Remove duplicate entries from the data  
- Save the cleaned and sorted output into a new file
