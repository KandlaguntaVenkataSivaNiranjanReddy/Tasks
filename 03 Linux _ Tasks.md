# Linux Lab - Tasks

---

## Task 21: System Performance Monitoring

### Objective
Monitor system performance and resource usage.

### Requirements
- Display all running processes  
- Identify processes that are using high CPU  
- Monitor system load in real time  

---

## Task 22: Network Connectivity Check

### Objective
Verify whether the system can connect to external networks.

### Requirements
- Send network requests to a remote server  
- Limit the number of requests sent  
- Analyze whether the response is successful or failed  

---

## Task 23: Real-Time Log Monitoring

### Objective
Monitor log files as they update in real time.

### Requirements
- Start monitoring a log file continuously  
- Filter and display only specific keywords  
- Verify that new entries appear instantly  

---

## Task 24: Hidden Files and System Visibility

### Objective
Understand how hidden files work in Linux.

### Requirements
- Create hidden files in a directory  
- List files normally and observe the difference  
- List all files including hidden ones  
- Identify different file types  

---

## Task 25: Data Transformation Pipeline

### Objective
Clean and transform data using command pipelines.

### Requirements
- Convert lowercase text into uppercase  
- Remove unwanted characters from the data  
- Save the cleaned data into a new file  

---

## Task 26: Advanced Log Debugging with Context

### Objective
Analyze logs by viewing surrounding lines for better understanding.

### Requirements
- Display lines that appear before an error  
- Display lines that appear after an error  
- Show full context around the error  

---

## Task 27: Permission Risk Analysis

### Objective
Identify files that have insecure permissions.

### Requirements
- Find files that have full access permissions (`777`)  
- Identify files that are read-only  
- Document all risky files and explain why they are unsafe  

---

## Task 28: Ownership Conflict Resolution

### Objective
Fix issues caused by incorrect file ownership.

### Requirements
- Create a file owned by one user  
- Try to access it from another user and observe failure  
- Change ownership to fix the issue  
- Verify that access is restored  

---

## Task 29: Disk Cleanup and Optimization

### Objective
Free up space by removing unnecessary files.

### Requirements
- Identify large files and directories  
- Delete unnecessary data  
- Verify that disk space has increased  

---

## Task 30: Process Debugging and Monitoring

### Objective
Analyze system processes and identify issues.

### Requirements
- Filter specific processes  
- Monitor resource usage  
- Identify any abnormal or high resource usage
