# Linux Lab - Tasks

---

## Task 41: Permission Failure and Recovery

### Objective
Understand and fix permission-related issues.

### Requirements
- Remove permissions from a file  
- Try to access it and observe failure  
- Restore correct permissions  
- Verify access is working  

---

## Task 42: Secure Script Execution

### Objective
Run scripts securely in a controlled environment.

### Requirements
- Restrict script access to owner  
- Execute the script  
- Verify permissions are secure  

---

## Task 43: Ownership Migration

### Objective
Transfer file ownership between users.

### Requirements
- Assign ownership to one user  
- Change ownership to another user  
- Apply changes recursively  
- Verify ownership  

---

## Task 44: Advanced File Search and Audit

### Objective
Perform deep file search for security analysis.

### Requirements
- Find empty files  
- Identify files with unsafe permissions  
- Document findings  

---

## Task 45: Directory Structure Validation

### Objective
Validate complex directory structures.

### Requirements
- Create nested directories  
- Count files and folders  
- Verify structure correctness  

---

## Task 46: Bulk File Creation and Classification

### Objective
Learn how to handle a large number of files and organize them based on their type.

### Requirements
- Create 20 to 30 files with different extensions such as `.txt`, `.log`, and `.conf`  
- List all files and group them based on their file type  
- Count how many files exist for each type  
- Verify that the classification is correct  
- Document your observations clearly  

---

## Task 47: Log Noise Filtering and Cleanup

### Objective
Clean log files by removing unnecessary entries and keeping only useful information.

### Requirements
- Create a log file containing `INFO`, `DEBUG`, and `ERROR` entries  
- Remove all `DEBUG` entries from the file  
- Keep only `INFO` and `ERROR` entries  
- Save the cleaned log into a new file  
- Verify that unwanted data is removed correctly  

---

## Task 48: Case Sensitivity and Search Behavior Analysis

### Objective
Understand how search behaves with different text cases.

### Requirements
- Create a file with mixed case words like `error`, `ERROR`, and `Error`  
- Perform a case-sensitive search and observe results  
- Perform a case-insensitive search and compare results  
- Document the difference between both outputs  

---

## Task 49: Pattern Extraction and Data Isolation

### Objective
Extract only useful information from large files.

### Requirements
- Create a file with repeated words or patterns  
- Extract only the matching words (not full lines)  
- Count how many times the pattern appears  
- Save extracted results into a new file  
- Verify the accuracy of the output  

---

## Task 50: Reverse Filtering and Data Exclusion

### Objective
Exclude unwanted data from logs and display only relevant information.

###  Requirements
- Create a log file with multiple entries  
- Display all lines except those containing `ERROR`  
- Perform case-insensitive exclusion  
- Show line numbers in the output  
- Verify that unwanted lines are removed
