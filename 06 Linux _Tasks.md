# Linux Lab - Tasks

## Task 51: Line Range Extraction for Data Analysis

### Objective
Work with specific sections of large files.

### Requirements
- Create a file with more than 100 lines  
- Extract a specific range of lines (e.g., 20–40)  
- Extract another range (e.g., 60–80)  
- Save extracted content into separate files  
- Verify accuracy of extracted data

## Task 52: Data Deduplication and Cleanup Pipeline

### Objective
Clean duplicate data and prepare a structured dataset.

### Requirements
- Create a file with duplicate entries  
- Sort the file content  
- Remove duplicate lines  
- Count the number of unique entries  
- Verify that duplicates are removed

## Task 54: Broken Symbolic Link Detection and Fix

### Objective

### Requirements

- Recreate or fix the symbolic link  
- Verify link functionality
