# Linux Lab - Tasks

---

## Task 11: Log Analysis using grep

### Objective
Learn how to search and filter important information from log files.

### Requirements
- Create a log file that contains different types of entries such as `INFO`, `ERROR`, and `WARNING`  
- Search and display only the lines that contain the word `ERROR`  
- Perform the same search without case sensitivity (`error`, `ERROR`, `Error`)  
- Display the line numbers along with the matched results  
- Count how many `ERROR` entries are present in the file  

---

## Task 12: File Statistics Reporting

### Objective
Understand how to analyze file data by checking its size and content statistics.

### Requirements
- Count the number of lines in a file  
- Count the total number of words in the file  
- Count the total number of characters  
- Display all these statistics together in a single command output  

---

## Task 13: Script Permission Setup

### Objective
Learn how to make a script executable and secure it properly.

### Requirements
- Create a simple script file  
- Add execute permission to the script  
- Restrict access so only the owner can run or modify it  
- Verify that the permissions are correctly applied  

---

## Task 14: Group-Based Access Control

### Objective
Manage permissions for a group of users working together.

### Requirements
- Create a new group in the system  
- Add multiple users to that group  
- Assign a file or directory to the group  
- Verify that users in the group have proper access  

---

## Task 15: File Linking (Hard & Soft Links)

### Objective
Understand how links work and how they behave when the original file changes.

### Requirements
- Create a file with some content  
- Create a hard link for that file  
- Create a soft (symbolic) link for the same file  
- Modify the original file and observe changes in both links  
- Delete the original file and observe what happens to both links  

---

## Task 16: Configuration File Editing Automation

### Objective
Automate changes in configuration files instead of editing manually.

### Requirements
- Create a configuration file with sample values  
- Replace a specific value in the file  
- Replace only the first occurrence of a value  
- Replace all occurrences of a value  
- Remove unwanted lines from the file  
- Save the updated content into a new file  

---

## Task 17: Log Processing Pipeline

### Objective
Process log files using multiple commands connected together.

### Requirements
- Extract specific log entries based on a keyword  
- Modify or transform the extracted data  
- Save the final processed output into a new file  

---

## Task 18: File Compression and Extraction (ZIP)

### Objective
Learn how to compress files and extract them when needed.

### Requirements
- Compress multiple files into a ZIP archive  
- Verify that the archive is created correctly  
- Extract the contents of the ZIP file  
- Confirm that all files are restored properly  

---

## Task 19: Archive Management using TAR

### Objective
Work with TAR archives for backup and storage.

### Requirements
- Create a TAR archive from a directory  
- Extract the contents of the archive  
- Verify that the extracted data matches the original  

---

## Task 20: Disk Usage Analysis

### Objective
Understand how disk space is used in the system.

### Requirements
- Check overall disk usage of the system  
- Display the size of directories  
- Identify which directory is using the most space
