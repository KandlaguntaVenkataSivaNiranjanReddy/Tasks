#  Linux Practice Tasks 

### Task 14: Group-Based Access Control
**Objective**  
Implement group-level permission management.

**Requirements**
- Create a group named devteam  
- Add user devops to the group  
- Assign group ownership of a file to devteam  
- Verify group assignment and access  

---

###  Task 15: File Linking and Dependency Handling
**Objective**  
Understand file linking mechanisms.

**Requirements**
- Create a file named original.txt  
- Create a hard link to the file  
- Create a soft link to the file  
- Delete the original file and observe behavior of both links  

---

### Task 16: Automated Configuration Editing
**Objective**  
Perform automated edits on configuration files.

**Requirements**
- Create a configuration file with sample values  
- Replace specific values globally  
- Replace only the first occurrence of a value  
- Remove specific unwanted lines  
- Save the modified configuration to a new file  

---

###  Task 17: Log Processing Pipeline Implementation
**Objective**  
Build a pipeline to process application logs.

**Requirements**
- Create a log file with mixed entries  
- Extract error-related entries  
- Modify extracted data (e.g., change keywords)  
- Save processed output into a new file
