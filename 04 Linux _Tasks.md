## Task 31: Backup and Recovery Workflow

### Objective
Ensure that data can be backed up and restored safely.

### Requirements
- Create backup of files or directories  
- Delete the original data  
- Restore the data from backup  
- Verify that the restored data is correct

## Task 32: Compressed Log Analysis

### Objective
Analyze application logs stored in compressed format without extracting them.

### Requirements
You are given a compressed log file (`app.log.gz`) from a production server.

- Search for all occurrences of the keyword `ERROR`  
- Display matching log entries  
- Count how many times the error occurred  
- Verify the results without extracting the file

## Task 35: SSH Access Configuration

### Objective
Enable secure and password-less SSH access between systems.

### Task
You need to configure SSH access from a client machine to a remote server.

- Generate SSH key pair on the client machine  
- Copy the public key to the remote server  
- Set proper permissions for SSH configuration files  
- Verify that login works without a password
